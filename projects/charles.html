Charles V/R Website

Frontend Development

/ /

Content Creation

This website is an immersive web narrative that provides users with historical background and context regarding the coronation of Holy Roman Emperor Charles V in Bologna, Italy, in 1530. It is the companion website to the virtual reality reconstruction of the coronation created by Greenhouse Studios.

I worked with another developer to design and code this website, using HTML, CSS, and JavaScript. We took an iterative approach to the design and development of this project, implementing feedback from the VR team to ensure the two projects came together cohesively.
