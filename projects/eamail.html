Eat and Move as I Like Video Game

Illustration

/ /

UX Design

/ /

Frontend Development

“Eat and Move as I Like” is an educational video game intended to teach middle school children how to make healthy food choices. It is featured in Heidi Karner’s Master’s Thesis titled “A Feasible and Tailored Nutrition Education Intervention for Middle-Schoolers.”

I redesigned the game’s graphics, creating original illustrations of various healthy snacks in each of the five food groups and a supermarket scene to serve as the backdrop to the game. I also implemented JavaScript code to improve the game’s user experience.
